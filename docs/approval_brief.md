# Project Approval Brief: NutriSnap AI

**STATUS**: APPROVED
**REASON**: 
This project aligns with the lab's goal of building AI-powered applications that solve real-world problems. It serves as an excellent technical validation for computer vision and frontend integration, and addresses a common local use-case for healthy living.

**ASSIGNED TO**: 
- **Lead**: MLOps Pod (Deployment/Frontend scaffolding)
- **Support**: ML Pod (Vision Logic & Prompt Engineering)

**SUCCESS METRIC**: 
A functional web application where a user can upload an image of food and receive:
1. Identification of the food item.
2. Estimated calorie count.
3. Macronutrient breakdown (Carbs, Protein, Fat).

**RISK FLAG**: 
Nutritional estimation is non-medical advice. The application **MUST** include prominent medical disclaimers to avoid liability regarding health and safety.
