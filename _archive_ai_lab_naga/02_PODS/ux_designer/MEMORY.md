## 🔁 Reusable UX Patterns
| Pattern | When to Use | Example |
|---|---|---|
| Greeting + quick options | When users don't know where to start | "Hi! I can help with: [A] [B] [C]" |
| Clarifying question | When input is ambiguous | "Did you mean [X] or [Y]?" |
| Graceful fallback | When bot doesn't understand | "I'm not sure I got that. Could you rephrase?" |
| Escalation | When user is frustrated | "Let me connect you with [Name]. [Contact info]" |
